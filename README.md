# LuckyEnchantment


## 自定物品
### 強化青金石
透過9個青金石磚合成，用於做一次隨機附魔（1 - 3個附魔）
### 超導青金石
透過9個強化青金石合成，用於保留現有裝備的附魔下，添加一個指定附魔
### 保留卷軸
只能透過購買獲得，用於保留一個指定附魔（可同時用最多4個）
### 消除卷軸
只能透過購買獲得，用於消除一個指定附魔（可同時用最多4個）
### 強化石
只能透過購買獲得，提升裝備等級<br>
當中，強化石成功機率將會按照等級高低不同而有成功、失敗、消失三個情況<br>
失敗會退還原本附魔裝備，消失則甚麼都不會歸還<br>

## 裝備等級
裝備等級分為未強化、+1、+2、+3、+4、+5、+6、+7、+8、+9<br>
等級高低將會影響裝備提供的buff/ 附魔効果的值<br>
如附了鋒利的劍在攻擊的時候，可以提升20%(未強化)/40%/.../200%最終攻擊值<br>
附了幸運的稿在挖掘礦物的時候，可以提升20%(未強化)/40%/.../200%最終掉落數量<br>

## 介面

### 主要操作介面（導航）

### 鍛造介面
透過放置裝備，並使用強化青金石/超導青金石來獲得附魔，使用保留卷軸來確保隨機附魔不會影響該指定附魔

### 消除介面
透過放置裝備，並使用消除卷軸來消除指定附魔

### 強化介面
透過放置裝備，並使用強化石來提升裝備的等級
